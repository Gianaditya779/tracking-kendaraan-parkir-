# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Form input kendaraan dengan validasi
- Tabel data dengan search & filter
- CRUD operations (Create, Read, Update, Delete)
- Auto increment nomor otomatis
- Rekap total Motor & Mobil
- Export data ke CSV
- Responsive design
- Real-time sync dengan Google Spreadsheet
- Google Apps Script integration
- REST API backend dengan Express.js
- React frontend dengan Vite
- Tailwind CSS styling
- Complete documentation

### Features
- ✅ Form input dengan dropdown jenis kendaraan
- ✅ Validasi plat nomor required
- ✅ Time picker untuk jam keluar/masuk
- ✅ Search berdasarkan plat nomor
- ✅ Filter berdasarkan jenis kendaraan
- ✅ Edit data inline
- ✅ Delete dengan konfirmasi
- ✅ Auto re-numbering setelah delete
- ✅ Export ke CSV
- ✅ Summary statistics (total, motor, mobil)
- ✅ Clean admin panel UI
- ✅ Blue & white color scheme
- ✅ Fully responsive

### Documentation

### Technical Stack
- Frontend: React 18 + Vite + Tailwind CSS
- Backend: Node.js + Express.js
- Database: Google Spreadsheet
- API: Google Apps Script Web App
